各考站主要不足点
1、病例分析
1.1首选胰岛素
1.2周围神经病变
1.32型糖尿病并糖尿病肾病V期
2、体格检查
2.1（口述）怒张：坐位或半坐位（45°）静脉充盈度超过正常
2.2钟型听诊器直接放在肿大的甲状腺上，听是否有静脉‘嗡鸣’音（口述），听是否有收缩期动脉杂音（口述）
2.3同一方法检查另一侧甲状腺
3、外科操作
3.1如果手术完毕，手套未破，又需连续施行另一手术时，可不用重新刷手，仅需用消毒液再浸泡或涂擦前臂。若前一次为污染手术或手套已破，则连续手术前必须重新外科洗手。
3.2用右手脱下左手手套至掌部，再以左手指脱去右手手套。最后用右手指在左手掌部推下左手手套。脱手套时，手套的外面不能接触皮肤，保护手臂及洗手衣不被手术衣外面污染。
3.3无菌手术衣包事先由巡回护士打开，无菌手套亦由巡回护士备好。
4、内科操作
4.1自皮肤至腹膜壁层用0.5%利多卡因逐层作局部浸润麻醉（每次注药前，必须回抽注射器栓，判断是否刺入血管）
4.2在穿刺点打皮丘（如条件限制不能打皮丘，可口述）
4.3严密观察有无出血和继发感染的并发症
5、急救操作
5.1连接呼吸机与面罩/鼻罩
5.2连接呼吸机管道及供氧管道
5.3检查，清除管道积水
6、病史采集
6.1G.病程中的一般情况 ：饮食、睡眠、精神、小便、体力体重（每项1分）
6.2F.诊治经过：接受过的检查、结果；有过哪些诊断；使用过的药物、剂量、疗程和疗效（每项1分）
6.3问诊结束时，谢谢患者合作
